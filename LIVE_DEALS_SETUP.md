# Live Deals Setup

NGKFlip reads a sanitized Amazon deals JSON file generated by RSForwarder. The NGKFlip GitHub repo should contain only website/backend code.

Do not commit Discord tokens, Amazon API secrets, `.env`, `node_modules`, or generated deal feeds.

## Canonical Flow

```text
RSForwarder on Oracle
        -> resolves final Amazon leads into RS-FS Live List
        -> exports sanitized JSON with export_ngk_amazon_deals.py
        -> NGKFlip reads DEAL_FEED_PATH
        -> /api/deals returns public deal JSON
        -> deals.html renders live cards
```

The browser never receives your Discord token, Google Sheets credentials, Amazon secret, or admin refresh key.

## Local Setup

Create a local `.env` file from `.env.example`:

```env
PORT=3000
DEAL_FEED_PATH=../RSForwarder/exports/ngk_amazon_deals.json
DEAL_REFRESH_MS=300000
MAX_DEALS=24
ADMIN_REFRESH_KEY=make_this_random
ACCESS_KEY=
SECRET_KEY=
PARTNER_TAG=leadflips-20
DISCORD_BOT_TOKEN=
DISCORD_CHANNEL_IDS=
DISCORD_MESSAGE_LIMIT=50
```

Then generate the feed from the parent `mirror-world` repo:

```powershell
py -3 -m RSForwarder.export_ngk_amazon_deals
```

Run NGKFlip:

```bash
npm install
npm start
```

Open:

```text
http://localhost:3000/deals.html
```

## Oracle Deployment

Recommended live setup:

- GitHub repo: `https://github.com/FlipGraphic/ngkflip`
- App path on Oracle: `/home/rsadmin/apps/ngkflip`
- Feed path on Oracle: `/home/rsadmin/bots/mirror-world/RSForwarder/exports/ngk_amazon_deals.json`

Server-only `.env`:

```env
PORT=3000

DEAL_REFRESH_MS=300000
MAX_DEALS=24

```

Deploy/update app code:

```bash
cd /home/rsadmin/apps/ngkflip
git pull --ff-only
npm ci
sudo systemctl restart ngkflip
```

Refresh the feed:

```bash
cd /home/rsadmin/bots/mirror-world
.venv/bin/python -m RSForwarder.export_ngk_amazon_deals
```

Verify:

```bash
curl -fsS http://127.0.0.1:3000/api/deals
curl -fsS http://127.0.0.1:3000/deals.html
```

## Optional Fallbacks

`DISCORD_BOT_TOKEN` and `DISCORD_CHANNEL_IDS` are optional fallback settings. Prefer the RSForwarder JSON feed so NGKFlip can stay separate from RS credentials.

Amazon Product Advertising API values are optional and only used by `/api/asin/:asin`. The main live deals feed does not need them.
